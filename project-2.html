Project Overview

WineGuru is a concept project that uses a finetuned OpenAI GPT-3 model with a custom dataset of wine reviews to generate wine recommendations based on user inputs such as wine type, price, and flavour profile.

WineGuru was built in a week in a team of 3 as part of the LabLab.ai OpenAI Stack Hackathon.
